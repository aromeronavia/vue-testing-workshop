# test-app

## Project setup
```
# Installs NVM
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.34.0/install.sh | bash
# Restart your terminal
nvm use 11.14.0
npm install
```

### Run the back end
```
node back/index.js
```

### Compiles and hot-reloads for development
```
npm start
```

### Run the tests
```
npm test
```
